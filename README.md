## 🛠 기술 스택

**Backend**
- Java, Spring Boot, Spring MVC
- Spring Interceptor (인증/인가 전역 처리)
- MySQL (메인 DB)
- Redis (휘발성 데이터 - 이메일 인증번호 등)

**Frontend**
- Thymeleaf, HTML, CSS, JavaScript
- 반응형 웹 디자인

**Infra**
- Linux, Nginx, AWS EC2 (단일 인스턴스)
- HTTPS (Let's Encrypt + OpenSSL)
- Crontab 기반 자동화 (백업, 디스크 점검)

## ✨ 핵심 기능

### 1. 프로그램 신청 & 시설(대관) 예약
- 센터 요구사항을 반영한 프로그램 신청 / 시설 대관 예약 기능 설계·개발
- 신청 시 **Word 템플릿의 placeholder에 데이터를 자동 매핑**하여 신청서 문서를 자동 생성
- 마이페이지에서 본인의 신청 내역 확인 가능
- 관리자 페이지에서 프로그램 등록·수정·삭제, 대관·신청 승인 여부 관리
- 전화·현장 접수 업무를 지원하는 **관리자 대리 신청 기능** 추가로 운영 효율 향상

### 2. 권한 관리 (Spring Interceptor)
- Interceptor로 관리자 / 일반 사용자의 접근 권한을 구분
  - `/admin` : 관리자 전용 경로로 서버단 검증
  - `/mypage` : 세션 기반 본인 확인으로 서버단 검증
- 공통 검증 로직을 전역으로 처리해 보안성과 유지보수성·확장성 동시 확보

### 3. Redis 기반 휘발성 데이터 관리
- 이메일 인증번호를 Redis에 저장
- 인증 완료 또는 만료 시 자동 삭제되도록 구현
- DB가 아닌 메모리 기반 저장소를 활용해 성능과 안정성을 동시에 확보

### 4. 이미지 업로드 최적화
- Summernote 에디터의 이미지 업로드에 **Thumbnailator** 적용
  - 최대 800px, 품질 70%로 자동 리사이징 및 압축
- **UUID 기반 파일명**으로 파일명 충돌 방지
- 임시 저장소(temp) 구조를 설계해 게시글 작성 중 업로드된 불필요한 파일이 남지 않도록 처리
- 결과적으로 서버 디스크 사용량 절감 및 업로드 성능 개선

### 5. 서버 운영 및 백업 체계
- 센터와 협업해 운영 서버 데이터를 센터 내부 PC로 주기적으로 백업하는 체계 구축
- **Crontab**을 이용해 자동화
  - DB 백업
  - 업로드 파일 백업
  - 디스크 사용량 점검
- 장애 발생 시 신속한 복구가 가능한 운영 체계 확보

### 6. 인프라
- HTTPS 적용 (Let's Encrypt + OpenSSL, 인증서 자동 갱신)
- Nginx 리버스 프록시
- AWS EC2 단일 인스턴스에서 DB·Storage 통합 운용으로 비용 절감
- 스냅샷 백업으로 운영 편의성 확보
